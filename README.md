## 组件世界是什么？

组件世界，英文名 WidgetStore，原来叫 NotionPet，是一个国产的**可嵌入式小组件库**。

它做的事情说起来很简单：给你一堆已经做好的小部件，你把链接复制到笔记软件里，嵌入进去，就可以显示天气、时钟、番茄钟、倒计时、热力图、每日古诗词……这些东西。

不需要写代码，不需要找第三方服务，国内服务器，打开就能用。

支持的平台挺全的：

- **Notion 系**：Notion、FlowUs、Wolai、ClickUp、Coda 等
- **双链笔记系**：Obsidian、思源笔记、Logseq、RemNote 等
- **其他**：WordPress、GitHub 主页、为知笔记……

基本上，只要你的编辑器支持"嵌入网页"，它就能进去。

## 它能塞进去什么东西？

按类型大概分这几块：

**基础小组件**——最常用的那批。天气、时钟、番茄钟、日历、倒计时、纪念日、习惯打卡热力图、白噪音……基本上效率类博主爱用的那些功能，这里面大多都有。
